# 智能体管理接口文档

本文档描述了 LingShu-AI 智能体（Agent）配置的增删改查（CRUD）接口。

## 1. 基础信息

- **基础路径 (Base URL)**: `/api/agents`
- **数据格式**: `application/json`

---

## 2. 数据模型 (AgentConfig)

| 字段名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `id` | Long | 唯一标识 ID (自增) |
| `name` | String | 唯一标识名 (英文字符，用于系统识别) |
| `displayName` | String | 显示名称 |
| `systemPrompt` | String (Text) | 核心系统提示词 |
| `factExtractionPrompt` | String (Text) | 事实提取提示词 (用于 GAM-RAG) |
| `behaviorPrinciples` | String (Text) | 行为原则 |
| `decisionMechanism` | String (Text) | 决策机制 |
| `toolCallRules` | String (Text) | 工具调用规则 |
| `emotionalStrategy` | String (Text) | 情感策略 |
| `greetingTriggers` | String (Text) | 招呼触发器 |
| `hiddenRules` | String (Text) | 隐藏规则 |
| `avatar` | String | 头像 URL 或 Base64 |
| `color` | String | UI 主题色 |
| `isDefault` | Boolean | 是否为系统默认智能体 |
| `isActive` | Boolean | 是否激活可用 |
| `createdAt` | LocalDateTime | 创建时间 |
| `updatedAt` | LocalDateTime | 更新时间 |

---

## 3. 接口列表

### 3.1 获取所有智能体
获取系统中配置的所有智能体列表。

- **URL**: `/api/agents`
- **Method**: `GET`
- **Response**: `List<AgentConfig>`

### 3.2 获取启用的智能体
仅获取当前状态为激活 (`isActive=true`) 的智能体。

- **URL**: `/api/agents/active`
- **Method**: `GET`
- **Response**: `List<AgentConfig>`

### 3.3 获取默认智能体
获取系统中被标记为 `isDefault=true` 的智能体。

- **URL**: `/api/agents/default`
- **Method**: `GET`
- **Response**: `AgentConfig` (若不存在则返回 404)

### 3.4 获取默认配置模版
用于新建智能体时预填充常用提示词。

- **URL**: `/api/agents/defaults`
- **Method**: `GET`
- **Response**: `AgentConfig` (只包含 Prompt 预设的模版对象)

### 3.5 获取指定智能体详情
- **URL**: `/api/agents/{id}`
- **Method**: `GET`
- **Parameters**: 
    - `id`: 智能体 ID
- **Response**: `AgentConfig`

### 3.6 根据 Name 获取智能体
- **URL**: `/api/agents/name/{name}`
- **Method**: `GET`
- **Parameters**: 
    - `name`: 智能体标识名
- **Response**: `AgentConfig`

### 3.7 创建智能体
- **URL**: `/api/agents`
- **Method**: `POST`
- **Body**: `AgentConfig` (JSON)
- **Response**: `AgentConfig` (带生成的 ID)

### 3.8 修改智能体
- **URL**: `/api/agents/{id}`
- **Method**: `PUT`
- **Parameters**: 
    - `id`: 智能体 ID
- **Body**: `AgentConfig` (JSON)
- **Response**: `AgentConfig` (更新后的对象)

### 3.9 删除智能体
- **URL**: `/api/agents/{id}`
- **Method**: `DELETE`
- **Parameters**: 
    - `id`: 智能体 ID
- **Response**: `200 OK`

### 3.10 设置为默认智能体
将指定智能体设为默认，系统会自动取消原默认智能体的标记。

- **URL**: `/api/agents/{id}/set-default`
- **Method**: `POST`
- **Parameters**: 
    - `id`: 智能体 ID
- **Response**: `AgentConfig` (更新后的对象)
