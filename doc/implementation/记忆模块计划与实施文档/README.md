## 一、项目概述

灵枢 (LingShu-AI) 的记忆模块是一个多层次、情感感知的认知存储系统。其核心目标是模拟人类的记忆机制，实现从短期会话到长期语义事实的转化，并通过 3D 可视化和治理后台提供全方位的记忆管理能力。

## 三、关键里程碑回顾

1. **里程碑 1**: 建立 Neo4j 与 PGVector 双基座存储，实现基础 RAG。
2. **里程碑 2**: 引入情感计算，将"死知识"转化为"带情绪的事实"。
3. **里程碑 3**: 完成 3D 图谱与治理后台，实现记忆的可视化治理。
4. **里程碑 4 (v3.3)**: 实现 GAM-RAG 检索路由机制，基于增益阈值智能选择图谱/向量检索策略。
5. **下一阶段目标**: 实现分段阶梯衰减算法与逻辑依存链管理，提升记忆生命周期管理的科学性和鲁棒性。
